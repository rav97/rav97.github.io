C#, Azure Service Bus, Entity Framework 6, PgSQL, NoSQL, xUnit, Bitmovin, React, Angular, Docker

POP PC

Aplikacja desktopowa umożliwiająca ręczne zarządzanie danymi z systemu Programu Ochrony Płatności. Widoki zaprojektowane w WPF w oparciu o wzorzec MVVM z wykorzystaniem stylów MaterialDesign dla XAML. Aplikacja wykonuje operacje CRUD w oparciu o logikę biznesową firmy co umożliwiło kompleksowe zarządzanie fakturami. Oprócz podstawowych mechanizmów CRUD aplikacja umożliwia import plików CSV, funkcje przypominania o zaplanowanych zadaniach, mechanizm kontroli aktualizacji, generowanie grafik certyfikatu uczestnicwa w programie POP oraz wiele innych. Rozwój aplikacji niejednokrotnie wymagał tworzenia customowych kontrolek, nadpisywania stylów i podobnych operacji "naginania" możliwości WPF do własnych potrzeb.

C#, XAML, MVVM, Entity Framework 6, MSSQL

POP Monitoring Service

Aplikacja usługi odpowiedzialna za automatyczną realizację harmonogramów (operacji monitoringu) na fakturach w systemie POP. Usługa, w elastycznie określanych godzinach dokonuje sprawdzania, czy na dany moment, dla spełnionych kryteriów dla faktur należy wykonać jakieś akcje (aktualizacja etykiety, przekazanie odpowiedniej wiadomości do wysyłki itp).

C#, Serilog, Entity Framework 6, XML

POP Import Service

Aplikacja usługi stanowiąca część integracji z większym klientem. Klient poprzez wiadomości email przekazuje pliki CSV z danymi nowo wystawionych faktur oraz obecnego stanu ich systemu. Usługa dokonuje importu oraz synchronizacji danych w celach poprawnego przebiegu procesu monitoringu.

C#, Serilog, Entity Framework 6, XML

POP API

Implementacja Web API umożliwiająca mniejszym klientom wygrywanie faktur do systemu POP oraz podastawowe zarządzanie nimi. Autoryzacja przy użyciu tokenów JWT oraz customowych nagłówków. Pozwala na wykonanie podstawowych operacji na fakturze za pominięciem kontaktu z operatorem systemu POP. API stanowi backend dla strony internetowej programochronyplatnosci.pl ale nie miałem udziału w jej tworzeniu.

C#, ASP.NET Core 3.1, Web API, Entity Framework Core, Serilog

Kontomatik Library

Biblioteka konsumująca BankingAPI oraz LendingAPI od Kontomatika. Zaprojektowana w taki sposób aby zminimalizować konieczność implementowania wielokrotnych asychnronicznych odwołań po stronie aplikacji. Metody w bibliotece mają odzwierciedlać metody z API w taki sposób aby poprzez jedno (oczekujące) odwołanie otrzymać żądaną odpowiedź. Następnie dzięki klasom wygenerowanym na podstawie udostępnianych XSD możliwa jest deserializacja odpowiedzi do postaci obiektów.

C#, HttpClient, XML, XSD, MSTest

Projekty osobiste

WPF MemeMaker

Aplikacja desktopowa służąca do tworzenia memów z wybranej grafiki. Na osobnej gałęzia opracowano rozwiązanie offline, oferujące podstawową funkcjonalność generatora memów. Pozwala na wgranie grafiki z komputera, dodanie pożądanej liczby podpisów, manipulację ich położeniem i rozmiarem oraz zapisaniem utworzonego mema na dysku. Rozwiązanie online oferuje taką samą podstawową funkcjonalność, ale także rozszeszone zostało o połączenie z zdalnym API (MemeMaker API), w celu uzyskania dostępu do bazy memów i szablonów przez nie udostępnianych.

C#, WPF (MVVM), XAML, HttpClient

MemeMaker API

Internetowe API umożliwiające zarządzanie bazą memów i szablonów. Utworzone do współpracy z aplikacją MemeMaker WPF (Online). Pozwala na dodawanie oraz pobieranie memów jak i szablonów, pobieranie list z informacjami o posiadanych grafikach, zapisywanie informacji o ich wykorzystaniu itp. Na osobnych gałęziach opracowano wersję działającą synchronicznie oraz asynchronicznie (wybór implementacji zależny od potrzeb). Całość została zaprojektowana zgodnie ze wzorcem Domain Driven Design oraz przetestowana przy użyciu testów integracyjnych xUnit.

C#, .NET 6, ASP.NET Core, MSSQL, EF Core, Swagger, Serilog, DDD Approach, xUnit

ExchangeRateAPI

Internetowe API stanowiące wartstę pośredniczącą z API Europejskiego Banku Centralnego. Z każdym nowym zapytaniem poszerza lokalną bazę danych w celu przyśpieszenia przyszłych zapytań o te same dane. Zapytanie składa się z podania par walutowych oraz okresu czasu. Dane te przekazywane są do API EXB, zwrócona odpowiedź zapisywana jest w lokalnej bazie API oraz jest konwertowana na pożądany format w celu jego zwrócenia.

C#, ASP.NET Core 3.1, MSSQL, EF Core, Swagger, Serilog, xUnit

Cosmic Rush

Prosta gra komputerowa z rodzaju Shoot'em Up, opracowana przy użyciu silnika Unity. Rozgrywka oparta jest o losowo pojawiające się bonusy oraz fale przeciwników, które należy kolejno pokonywać zdobywając za to punkty rankingowe. Po przekroczeniu określonych progów punktowych na planszy pojawia się silniejszy przeciwnik - boss którego pokonanie stanowi większe wyzwanie.

Unity 2D, C#, JSON, XML

Be My Brush

Aplikacja umożliwiająca rysowanie obrazów za pomocą ruchów ciała. Użytkownik przy użyciu specjalnie skonstruowanego wskaźnika - rękawiczki. Ma możliwość malowania obrazów przy wykorzystaniu ruchów własnego ciała przechwyconych w obrazie z kamery. Interfejs aplikacji przystosowany został do obsługi na odległość.

Python 3.6, OpenCV

Park Sensors

Projekt i implementacja systemu czujników parkowania z wykorzystaniem mikrokontrolera STM32F4 Discovery oraz czujników odległości HC-SR04. Projekt zakładał przygotowanie prototypu urządzenia umożliwiającego ocenę odległości między pojazdem a potencjalną przeszkodą znajdującą się w martwych punktach strefy cofania. Odległość od przeszkody sygnalizowana była przy użyciu wyświetlacza oraz sygnałów dźwiękowych.

C, STM32, Standard Peripheral Library for STM
